# Project Overview:

The E-Commerce Grocery Website is a full-stack web application designed to provide users with a convenient platform for browsing, selecting, and purchasing grocery items online. Built using Python (Flask/Django), HTML, CSS, JavaScript, and MySQL, the project focuses on creating a seamless shopping experience with features like product listing, cart management, and secure checkout. The application is deployed on AWS Cloud, utilizing EC2 for hosting, S3 for media storage, and RDS for database management to ensure scalability, reliability, and high availability. Docker was used for containerization and Git for version control, enabling efficient development, deployment, and maintenance.
